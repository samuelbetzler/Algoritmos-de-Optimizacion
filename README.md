# Algoritmos de Optimización

Portafolio personal de la asignatura **Algoritmos de Optimización**. En este repositorio se recopilan las actividades guiadas, proyectos (trabajos prácticos) y el código desarrollado como respuesta a las actividades de clase.

## Autor

**Samuel Segundo Campozano López**

## Descripción

Este repositorio funciona como un portafolio académico donde se irá subiendo, de forma progresiva, todo el material de la asignatura: enunciados, soluciones, código fuente y notas. El objetivo es mantener un registro ordenado del avance a lo largo del curso.

## Contenido

- Resolución de ejercicios y problemas planteados en clase.
- Implementaciones de algoritmos de optimización.
- Notas y material de apoyo.

## Estructura del repositorio

```
Algoritmos-de-Optimizacion/
├── ACTIVIDADES GUIADAS/       # Soluciones a las actividades guiadas de clase
├── PROYECTO (Trabajo Práctico)/ # Trabajos prácticos y proyectos
├── notas/                     # Apuntes y material de apoyo
└── README.md                  # Este archivo
```

## Tecnologías

- Lenguaje principal: Python
- Librerías y herramientas según cada ejercicio.

## Cómo usar este repositorio

Cada carpeta contiene las soluciones correspondientes a una actividad o tema. Para ejecutar el código, clona el repositorio:

```bash
git clone https://github.com/samuelbetzler/Algoritmos-de-Optimizacion.git
```
